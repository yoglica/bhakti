Next.js SEO Basics: Getting Started with Metadata

Published on June 8, 2026 • 3 min read

What is Next.js Metadata API?

The Next.js App Router introduced a powerful Metadata API that makes SEO implementation straightforward. You can define metadata at the page level or layout level.

Basic Metadata Setup

Here's a simple example of metadata in your layout.js file:

export const metadata = {
  title: 'My Blog',
  description: 'A blog about web development',
}

Dynamic Metadata for Pages

For blog posts, you can generate dynamic metadata based on the content:

export async function generateMetadata({ params }) {
  const post = await getPost(params.slug)
  return {
    title: post.title,
    description: post.description,
  }
}

Conclusion

Next.js makes SEO simple with its built-in Metadata API. Start by adding titles and descriptions to your pages, then expand to Open Graph and Twitter cards.
